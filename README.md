# MÁQUINA DE COMPRAS DE UM MERCADO
Esse sistema consiste em uma máquina que:
- recebe a tabela diária de preços do mercado
- calcula o valor total da compra de cada cliente
- armazena o total de cada um deles em um relatório diário

Mesmo que, em um contexto real, essas atividades sejam mais complexas, um programa como esse contempla os principais aspectos do funcionamento de um mercado.

## O que foi praticado
- Uso de uma função
- Uso de um dicionário
- Tratamento de erros
- Estruturas condicionais e de repetição
- Manipulação de arquivo

## Tecnologias
- Linguagem: Python
- Ferramentas: Nenhuma

## Estrutura
O código foi separado em dois setores:

- main.py: contém a interface do sistema, responsável pela maior parte dos
processos (cálculo de valores, interação com o usuário e manipulação de arquivos).

- tabela_diaria.py: responsável por criar o dicionário contendo os        produtos disponíveis e seus respectivos preços.
